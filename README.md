# Absence Manager

# Description
This app manages the list of absences with filterig option 

### Server

#### Installation
```bash
cd server 
npm install 
npm start
```

The server will be started on http://localhost:4000

#### Build

```bash
npm run build
```

### Client

#### Installation
```bash
cd client 
npm install 
npm start
```


The client will be started on http://localhost:3000

#### Testing

```bash
npm test
```

#### Build

```bash
npm run build
```
